# Deploy to Vercel (Recommended - 5 Minutes!)

## Why Vercel?

✅ **Made for Next.js** - Official hosting platform for Next.js  
✅ **Instant deployment** - Takes ~2-3 minutes  
✅ **Free tier** - No billing setup needed  
✅ **Auto HTTPS** - SSL enabled by default  
✅ **Global CDN** - Automatic worldwide distribution  
✅ **Environment variables** - Supabase credentials supported  
✅ **One command deploy** - `vercel` and you're done  

---

## Quick Deploy to Vercel

### Step 1: Install Vercel CLI
```powershell
npm install -g vercel
```

### Step 2: Navigate to Project
```powershell
cd C:\Users\smbal\silicon-boardroom
```

### Step 3: Deploy
```powershell
vercel
```

### Step 4: Follow Prompts
- Link to Vercel account (or create one)
- Confirm deployment
- Set environment variables if asked
- Done! 🎉

---

## What Happens

1. **Vercel CLI** uploads your code
2. **Vercel builds** your Next.js app
3. **Vercel deploys** to their global network
4. **You get a URL** like: `silicon-boardroom.vercel.app`

**Time:** ~2-3 minutes

## Set Custom Domain

After deployment:
1. Go to https://vercel.com/dashboard
2. Select your project
3. Go to Settings → Domains
4. Add your domain
5. Update DNS records
